# Project Details

### Theme of site / Aesthetics
- "Curating experiences and environments"
- A lifestyle, curating experiences, travel, hosting, inspo tips, DIY projects, Home decor website
- This will be a blog style website that also has a list of products that can be purchased.
- Offer some themes and colors for the site and let me pick.

### Features
- A login feature that allows the admin to login and manage the site
- Users should be able to create an account to place orders, view past orders and see tracking info.
- Guests should be allowed to purchase items without making an account if they want but they won't have an account to track past orders as a warning.
- The admin should be able to create items to sell, create blog articles with images, links and formatting.
- For products, the admin should be able to add/remove and modify different fields such as image of product, description, name, size option drop down, and other custom fields. Like if we wanted to add a "Color" text box that users could put in their preferred color. Or drop down options.
- The admin should be able to post blogs.
- There should be a menu option hamburger style that can expand so users can navigate the site.
- The admin should be able to select a specific blog as a "featured blog" that shows up on the front page.
- Visitors of the site should be able to search blog posts, and view all of them on a full listing page from the hamburger menu. They should also be able to sort by Oldest/Newest etc. 
- The site should allow the admin to upload images and files for viewing.
- There should be at a minimum a Blog tab, Products tab, and a Free Guides tab
- The admin page should be located at /admin
- 2FA should be required for the admin user. It needs to be set up on the first login.
- The site should work on web and mobile.
- The admin should be able to mark certain orders as complete/fulfilled/in-progress, and the admin should be able to select orders to export to a spreadsheet. The spreadsheet should separate out Name, Address Line1, Address Line 2, City, State, Zip code. This will allow for easy label creation in Pirate Ship.
- users should get an email confirmation when their order is confirmed, or shipped.
- The site should allow the admin to upload tracking numbers to the site, and depending on the shipper, the site should generate a link to the tracking number on the site (FedEx, UPS, USPS)


### Technology Requirements
- Stripe integration
- Vercel hosted
- Look at any other technologies required
- 2FA code

### Review Requirements
- QA agent that reviews for usage bugs
- Design agent to make sure the site looks beautiful
- A security review agent to ensure the site is secure.
